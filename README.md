\# IronsSableGuard



Compatibility and crash-fix mod focused on making

Irons Spells 'n Spellbooks work correctly with Aeronautics / Sable AP environments.



\## Features

\- Fixes crashes caused by spell teleportation inside AP worlds

\- Prevents problematic spatial spell interactions

\- Handles compatibility between:

&#x20; - Irons Spells 'n Spellbooks

&#x20; - Aeronautics

&#x20; - Sable



\## Environment

\- Minecraft 1.21.1

\- NeoForge



\## Current Work

\- Portal interaction fixes

\- AP-safe spell handling

\- Experimental teleport protection

Experimental / Work in progress
